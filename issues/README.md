# issues/

Mission issues are tracked on GitHub Issues, not in this directory.

This directory exists for archived mission summaries cross-linked from `.ai/decision-log.md` once the GitHub issue closes - one short markdown file per closed mission, summarizing the merged PRs, the governor outcome, and any follow-up commitments. Skip if the decision-log entry already captures everything.

If this directory still has no real entries six months in, retire it.
